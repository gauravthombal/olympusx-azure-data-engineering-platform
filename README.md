#  OlympusX – End-to-End Azure Data Engineering Platform

## Overview

OlympusX is a cloud-based Azure Data Engineering project that demonstrates the complete lifecycle of an analytics pipeline using Olympic datasets.

The project ingests raw data from GitHub into Azure Data Lake Storage Gen2, transforms it using Azure Databricks and PySpark, stores curated data in Azure Synapse Analytics, and visualizes insights using Power BI.

## Project Workflow

1. Olympic datasets stored in GitHub
2. Azure Data Factory copies data to Data Lake
3. Databricks processes and transforms data
4. Curated data loaded into Azure Synapse
5. Power BI connects to Synapse for reporting
